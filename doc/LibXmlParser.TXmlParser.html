Описание

Поля

FBuffer: PChar;

— Internal Properties and Methods

FBufferSize: INTEGER;

NIL if there is no buffer available

FSource: STRING;

0 if the buffer is not owned by the Document instance

FXmlVersion: STRING;

Name of Source of document. Filename for Documents loaded with LoadFromFile

FEncoding: STRING;

XML version from Document header. Default is '1.0'

FStandalone: BOOLEAN;

Encoding from Document header. Default is 'UTF-8'

FRootName: STRING;

Standalone declaration from Document header. Default is 'yes'

FDtdcFinal: PChar;

Name of the Root Element (= DTD name)

FNormalize: BOOLEAN;

Pointer to the '>' character terminating the DTD declaration

EntityStack: TEntityStack;

If true: Pack Whitespace and don't return empty contents

FCurEncoding: STRING;

Entity Stack for Parameter and General Entities

Elements: TElemList;

Returns document buffer — DTD Objects

Entities: TNvpList;

Elements: List of TElemDef (contains Attribute Definitions)

ParEntities: TNvpList;

General Entities: List of TEntityDef

Notations: TNvpList;

Parameter Entities: List of TEntityDef

CurPartType: TPartType;

Clear Document — Scanning through the document

CurName: STRING;

Current Type

CurContent: STRING;

Current Name

CurStart: PChar;

Current Normalized Content

CurFinal: PChar;

Current First character

CurAttr: TAttrList;
